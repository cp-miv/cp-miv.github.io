Egalité d'image

Introduction

Cette démonstration permet de calculer l'empreinte MD5 d'images.
Si les valeurs calculées sont identiques, cela doit indiquer que les images sont les mêmes.

Instructions

Téléchargez le pack d'images ci-dessous et réalisez son extraction.
Placez ensuite les différentes images dans les emplacements ci-dessous pour calculer l'empreinte MD5.

Essayez différentes combinaisons pour voir les résultats !
