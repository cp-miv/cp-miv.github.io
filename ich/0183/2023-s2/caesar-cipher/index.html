Caesar Cipher : Chiffrement par décallage

Introduction

Cette application permet d'appliquer le Code de César; chiffrement par décallage, sur un message afin de le chiffrer ou le déchiffrer.
L'objectif est que les deux parties (personne, application) parviennent à échanger un contenu textuel sans que celui-ci ne transite en clair.
Pour ce faire, chaque lettre du message est décallée d'une certain nombre de position dans l'alphabet selon une clé connue.

Instruction

Deux personnes sont nécessaires pour utiliser cette application : une pour chiffrer le message, l'autre pour le déchiffrer.
Répartissez-vous ces rôles, puis sélectionnez le bouton correspondant à votre mission.

Chiffrer un message

Votre rôle est de chiffrer un message et le transmettre au destinataire.

Déchiffrer un message
