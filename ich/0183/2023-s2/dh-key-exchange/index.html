Diffie-Hellman : Echange de clés

Introduction

Cette application permet de s'accorder sur un numéro secret à l'algorithme d'échange de clé Diffie-Hellman.
L'objectif est que les deux parties (personne, application) parviennent à avoir la même clé, sans que celle-ci ne soit échangée.
En lieu et place, elle est calculée à l'aide de valeurs transmises.

Instruction

Deux personnes sont nécessaires pour utiliser cette application : une pour initier l'échange, l'autre pour le rejoindre.
Répartissez-vous ces rôles, puis sélectionnez le bouton correspondant à votre mission.

Votre rôle est d'initier l'échange de clés avec un destinataire.
